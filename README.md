# Test S3 and EC2

Projeto de estudo utilizando AWS S3 e EC2 com Spring Boot.

## 🚀 Tecnologias
- Java
- Spring Boot
- AWS S3
- AWS EC2
- Maven

## 📌 Objetivo
Criar uma API capaz de realizar upload e acesso a arquivos armazenados no Amazon S3, rodando em uma instância EC2.

## ⚙️ Funcionalidades
- Upload de arquivos para o S3
- Recuperação de arquivos
- Integração com AWS SDK

## ▶️ Como executar
```bash
mvn spring-boot:run
